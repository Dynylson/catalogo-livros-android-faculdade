# Catálogo de Livros

Aplicação acadêmica para gerenciar um catálogo pessoal de livros, com autenticação, cadastro, edição e organização de leituras. O objetivo é oferecer uma experiência simples para acompanhar títulos, autores, gêneros, status de leitura e avaliações.

## Objetivo da aplicação

- Centralizar o registro de livros do usuário
- Facilitar criação, edição, visualização e exclusão de livros
- Acompanhar status de leitura e avaliações
